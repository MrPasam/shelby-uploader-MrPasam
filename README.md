# Shelby Storage Uploader

A lightweight experimental tool for uploading files to the Shelby decentralized storage network.

---

## Overview

Decentralized storage systems are becoming an important part of Web3 infrastructure.

Shelby provides a distributed storage network designed for scalability and developer accessibility.

This project experiments with basic file upload workflows to better understand how developers can interact with Shelby through simple scripts and automation tools.

The main focus of this repository is learning, experimentation, and building lightweight developer tools.

---

## Project Goals

• Explore Shelby decentralized storage workflow  
• Build a minimal file uploader tool  
• Document experimentation results  
• Create simple developer examples  

---

## Features

• Lightweight uploader script  
• Simple developer workflow  
• Experimental decentralized storage testing  
• Easy integration for Web3 developers
